## 💡 사용 가능한 기능

### 즉시 사용 가능
1. **타임라인**: 모든 알림 보기, 필터링, 읽음 처리
2. **카테고리**: 7개 카테고리별 알림 분류
3. **통계**: 시간대/요일/카테고리/앱별 차트
4. **학습**: 알림에서 외국어 추출, 학습 아이템 관리
5. **설정**: 권한 관리, 데이터 삭제, 정보 확인

### 데이터 흐름
```
알림 수신 (Android)
    ↓
자동 분류 (Classifier)
    ↓
DB 저장 (Drift)
    ↓
타임라인 표시
    ↓
학습 아이템 추가 (선택)
    ↓
통계 생성
```
